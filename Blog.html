Apresentação

Início

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos. Neste curso será apresentado as linguagens HTML, CSS e Javascript. Abaixo apresentaremos com mais detalhes o passo a passo.

Ferramentas utilizadas

Início

No decorrer do curso, serão utilizadas ferramentas como VSCode*. Que é um editor de código open source (código aberto) desenvolvido pela Microsoft. A saber, ele está disponível para Windows, Mac e Linux. É criado com Electron, ferramenta criada pelo GitHub que permite a criação de softwares Desktop com HTML, CSS e JavaScript. Faça o download do aplicativo e instale-o e inicie-o. Precisaremos também de uam extensão chamada Live Server, que é utilizada para mostrar a página que está sendo desenvolvida em tempo real.

Inspecionador de elementos

Início

O inspecionador de elementos é um utilitário que permite a vizualização do código-fonte subjacente de qualquer página Web.

É uma ferramenta extremamente útil, pois permite você testar uma alteração ou diagnosticar um problema, serve também, como base de estudos e entendimento, permitindo algumas alterações temporárias na plataforma e o entendimento de como o código funciona.

Estrutura básica do HTML

Início

Por ser um padrão, utilizamos a inscrição "index.html" para iniciar um projeto no VSCode. Pode ser nomeado com qualqeur nome mas por boas práticas mantemos no padrão. Podemos usar um atalho para criar a base da estrutura, que é: "html:5". Nela já vamos ter um padrão de cabeçalho e de estrutura inicial.

Tags

Início

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página. As tags possuem uma característica principal que é os sinais estarem dentro dos sinais de maior e menos "<>".

Atributos das Tags

Início

Os atributos HTML são palavras especiais dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc.

Textos

Início

Temos os atributos de texto que são os Hs para poder definir os títulos p define os parágrafos, sup para definir um símbolo como um expoente, a função sub coloca as palavras como base do texto.

Listas ordenadas e não ordenadas

Início

Listas ordenadas são listas tem como sequência um numeral. Já as não ordenadas são apresentadas como um marcador de sequencia.

Links

Início

No caso dos links, usamos um ancorador "a" junto da tag "href" para conseguir linkar, seja para redirecionar para outro site, podendo ser em uma nova aba ou nova janela. também serve para navegar no próprio site.
